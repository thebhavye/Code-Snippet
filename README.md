# SnippetVault

A personal code snippet library that lives entirely in your browser. Save, tag, search, and reuse your best code — no account, no server, no setup.

## Features

- **20+ languages** with live syntax highlighting
- **Instant search** across title, tags, description and code
- **Smart code editor** with tab indent, auto brackets and Enter-key indentation
- **One-click copy** for any snippet
- **Tag & filter** by language or custom tags
- **100% local** — all data saved in localStorage, completely private
